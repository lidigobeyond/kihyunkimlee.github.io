ts-node Error: Cannot find module '@types/node/package.json'

작성일 2021-06-10 Edited on 2021-06-11 In Do it! typescript 프로그래밍 Views: Disqus:

에러 발생

typescript를 설치하고 ts-node를 설치한 뒤에 ts-node를 실행시켜보니 아래와 같은 에러가 발생했다.

발생 원인

ts-node를 실행시키기 위해서는 @types/node라는 패키지가 필요한 거 같다.
@types/node 패키지를 설치해줬다.

해결

@types/node 패키지를 설치하니 ts-node가 정상적으로 동작한다.

(근데 왜 패키지가 아니라 ‘package.json’ 파일이 없다고 나오는지 잘 모르겠다…)
